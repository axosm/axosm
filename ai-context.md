This is a Browser MMO (Massively Multiplayer Online) RTS (Real-Time Strategy) game inspired by classics like Ogame or Tribal Wars. The game also take inspiration from games like Civilization and Stellaris.



Game is tick based instead of turn based.

Players start on planet and play like civ. players can build on tiles (one building per tile), build storage, mines, defenses and more. players can do research. ennemy units on same tile fight. unit and battles takes time. players can build space ships and send them to space. there are no tiles in space and ships move to coordinates. there are asteroids. players can build station or stargate, transport small ships or troops (on planet as well), cloak ships, scan other ships. Ships can also be in orbit (high and low). players can join alliances, declare war. help me build a schema for my database. i start with sqlite (local/dev) but i want to support posgres (prod) as well.
